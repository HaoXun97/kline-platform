# K 線圖分析平台

## 專案簡介

提供台股、美股和加密貨幣的 K 線圖查詢的網頁平台。使用者可以方便地搜尋並查看各種金融市場的歷史走勢圖及技術指標。

## 功能特點

- 支援台股及美股市場股票查詢
- 支援加密貨幣行情查詢
- 提供 K 線圖技術分析工具
- 簡潔的搜尋介面
- 即時的下拉式搜尋建議
- 完整的技術指標支援(待完成)

## 如何使用

1. 選擇市場類別（美股/加密貨幣）
2. 在搜尋框輸入股票代號或幣種名稱
3. 從下拉選單中選擇想要查看的標的
4. 查看 K 線圖及相關技術指標
